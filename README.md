# pageController
一个类似腾讯新闻的分页控制器，支持标题的增删，标题联动等。首页是嵌套collectionview,分类选择控制器是http://blog.csdn.net/u013282507/article/details/54374952 在这位大哥的基础上修改的，支持拖动排序。简书地址：http://www.jianshu.com/p/b982c38047e1
